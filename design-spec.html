Listing Status Pill Consolidation

Consolidating Active, Coming Soon, and Zillow Preview status filters into a single pill on the SRP pill bar — matching the existing "Listing Type" pill pattern.

Platforms: iOS, Android, Voyager Surface: Search Results Page Date: April 2026
Context
Why we're doing this
The current filter architecture treats listing statuses as listing types, making it impossible to isolate pre-market inventory.

! The Problem

There is no way to toggle "Active" OFF. When a user checks Coming Soon or Zillow Preview, 19,000+ active listings still flood the results. The filter doesn't do what it says.

✓ The Solution

Introduce an explicit Active status checkbox. Consolidate Active, Coming Soon, and Zillow Preview into a "Listing status" pill on the SRP bar — enabling true status-based isolation.

● Success Metrics

  • % of users who uncheck Active after finding the new filter
  • Click-through rate on Preview listings with Active OFF vs. ON
  • Engagement lift: saves, shares, contact-agent actions on Preview listings

♦ Scope

  • New "Listing status" pill on SRP pill bar
  • Bottom sheet with Active / Coming Soon / Zillow Preview checkboxes
  • Pill auto-hides when all three are ON (default)
  • Two label options under evaluation

Bottom sheet specifications

1
Drag handle — Small centered bar at top of sheet. Standard iOS/Android bottom sheet affordance.
2
Header row — "Listing status" title left-aligned, "Done" action link right-aligned in Zillow blue.
3
Active checkbox — New addition. Checked by default. Unchecking removes 19K+ active listings from results, enabling pre-market isolation.
4
Coming Soon — Existing filter, relocated from Listing Type into this status group. Info icon links to tooltip.
5
Zillow Preview — New filter with yard-sign icon and NEW badge. Enables isolated preview-only inventory view.
6
Real-time updates — Checking/unchecking immediately updates the pill label, result count, and listing cards behind the sheet overlay.
7
Dismiss behavior — Tap "Done", tap the overlay, or swipe down. The X on the pill resets all three to ON (default), hiding the pill.
